## Packages

- React Navigation
- Expo Location
- Expo Permission
- KeyboardAwareScrollView
- React Native Maps
- Axios

## Running the project

Clone this repository :

```
git clone https://github.com/LilaGuill/airbnb-frontend.git
cd airbnb-frontend
```

Install packages :

```
npm install
```

or

```
npm install
```

When installation is complete, run the project with:

```
expo start
```
